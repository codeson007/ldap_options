工具使用场景：

   利用openldap接管linux客户端，进行用户登录、验权、获取基础信息、密码管理策略等。适合处理“统一用户”认证的同学使用，如果你被openldap搞得焦头乱额，请试试吧。

工具主要功能：

           1,添加删除用户、组

           2,添加删除组内用户

           3,管理密码策略

           4,锁定、解锁用户
